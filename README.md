# SolarTracker
Build a solar tracker with 2 DOF that follows the sun, based on Arduino Uno.

This project is based on the following project by GreatScottLab:

https://www.instructables.com/DIY-Miniature-Solar-Tracker

and the 3D print files by fbuenonet:
https://www.thingiverse.com/thing:708819

My humble contribution:
I added some step-by-step instructions that seem be suitable for a 5th year science STEM class in Flanders, Belgium.
And I rewrote the Arduino code to be somewhat easier to read.
